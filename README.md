# Red Wine Quality Analysis in Tableau

## Overview

This project explores how chemical characteristics relate to red wine quality ratings using interactive Tableau dashboards. The analysis focuses on understanding how multiple variables interact to influence perceived wine quality rather than identifying a single “perfect” wine profile.

The project uses the Red Wine Quality dataset from Kaggle/UCI and applies Tableau dashboards, LOD calculations, dashboard actions, and comparative visualizations to analyze relationships between wine chemistry and wine ratings.

## Dashboard 1 – Understanding Wine Quality

### Features

* Interactive quality selector
* Scatter plot visualization
* LOD trend analysis
* Dashboard filtering actions

### Key Visualizations

#### Scatter Plot

* Alcohol vs. Volatile Acidity
* Color represents wine quality
* Size represents sulphates

This visualization demonstrates that wine quality is influenced by multiple variables and that wines with similar quality ratings can still exhibit very different chemical profiles.

#### LOD Trend Charts

The dashboard includes Level of Detail (LOD) calculations to show average trends across wine quality levels.

Observed trends include:

* Higher-quality wines generally contain higher alcohol levels
* Higher-quality wines generally show lower volatile acidity
* Sulphates exhibit a moderate positive relationship with quality

### Key Takeaways

* Wine quality depends on multiple interacting characteristics
* High-quality wines can exhibit different chemical profiles

## Dashboard 2 – What Differentiates Wine Quality Most?

### Features

* Advanced dumbbell chart visualization
* Normalized attribute comparison
* Comparative analysis between lower- and higher-quality wines

### Dumbbell Chart Analysis

The dumbbell chart compares normalized chemical characteristics between Quality 4 and Quality 7 wines.

Normalization was applied because wine characteristics exist on different numerical scales. This allows variables such as alcohol, sulphates, and volatile acidity to be compared fairly within the same visualization.

The greater the distance between the two points, the more strongly that characteristic differentiates lower- and higher-quality wines.

### Key Insights

* Certain chemical characteristics show much larger differences between lower- and higher-quality wines
* Alcohol and volatile acidity demonstrated some of the strongest separation between quality groups
* Some variables showed relatively small differences despite changes in wine quality
